# TalentMind AI — Candidate Ranking Hackathon

This repository contains a Django + DRF backend and a React + Tailwind frontend for a candidate ranking system powered by semantic retrieval, feature engines, and an LLM-based recruiter agent.

Key features
- Semantic retrieval with embeddings and FAISS (fallback to in-memory cosine)
- Skill, Experience, Behavioral scoring engines
- Gemini LLM Recruiter Agent integration (fallback when API unavailable)
- Hybrid ranking combining semantic, structured scores, and LLM signals
- Explainability with per-candidate text explanations
- Frontend: React + Tailwind dashboard for JD upload, ranking, explainability
- Export: CSV exports for ranked candidates and submission-ready formats

Quickstart (development)

1. Backend

```bash
cd backend
python -m venv .venv
.\.venv\Scripts\activate
pip install -r ../requirements.txt
python manage.py migrate
python manage.py runserver
```

2. Frontend

```bash
cd frontend
npm install
npm run dev
```

Environment variables
- `GEMINI_API_KEY` — (optional) API key for Google Gemini generative models. If not set, the system uses a deterministic fallback for recruiter analysis.

Files of interest
- `backend/api/services/gemini_agent.py` — LLM integration and fallback logic
- `backend/api/services/ranking.py` — hybrid ranking logic and explainability
- `frontend/src` — React dashboard
- `backend/exports` — sample CSV exports generated by `backend/scripts/generate_sample_exports.py`
